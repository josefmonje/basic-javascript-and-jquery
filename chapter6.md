## Control Flow


### If Statements

```
if (expression evaluates as true) {
  doSomething()
}

if (true) {
  doSomething()
} else {
  doSomethingElse()
}

if (true) {
  doSomething()
} else if (anotherExpression) {
  doAnotherThing()
} else {
  doSomethingElse()
}
```
> *If ...else Statements*
